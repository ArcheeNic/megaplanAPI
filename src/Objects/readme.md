## Инструменты для импорта
1. Корпируется информация из таблицы
2. Вставляется в Php файл
3. Выполнеяется замена регулярным выражением 

	с
	```regexp
	^[\s]*([\S]+?)[\s]+([\S]+)[\s]+([\S\s]*?)$
	```
	на
	```regexp
	/\*\*\n\* @var $2 $3 \n\*/\npublic \$$1;
	```
